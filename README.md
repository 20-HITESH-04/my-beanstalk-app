# AWS Elastic Beanstalk Node.js Deployment

This project demonstrates automated CI/CD deployment of a Node.js application using:

- **GitHub**
- **GitHub Actions**
- **AWS Elastic Beanstalk**
- **Node.js + Express**

---

## Features
- Modern UI for all routes
- API endpoint `/api/info`
- Health endpoint `/health` used by AWS
- 404 custom UI page
- Fully automated AWS deployment
- Logging using Morgan

---

## Tech Stack
- Node.js
- Express.js
- GitHub Actions (CI/CD)
- AWS Elastic Beanstalk

---
